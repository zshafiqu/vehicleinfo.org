The vehicleinfo.org API

This project exposes a public API endpoint that can be used to get data on any vehicle(s) a client may need. Implementing the API is as simple as making an HTTP request to:

https://vehicleinfo.org/api/

and specifying the resource you would like.

GET /year

The first route accepts GET requests for a specific year within range, and the returns a JSON response of every vehicle for that year, including its individual attributes.

Try this route by pressing this link

www.vehicleinfo.org/api/2002/

GET /year/make

The second route for the API accepts GET requests for a specific year and make, and returns a JSON response of all vehicles produced by that manufacturer in that year.

Try this route by pressing this link

www.vehicleinfo.org/api/2002/acura

GET /year/make/model

The final route accepts GET requests for a specific year, make, and model resulting in a JSON response of the attributes for that specific vehicle.

Try this route by pressing this link

www.vehicleinfo.org/api/2002/acura/tl
