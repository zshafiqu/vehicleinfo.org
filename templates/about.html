About Me

I am currently a fourth year student at the University of California, Santa Cruz where I study computer science. Motivation for this project came from the fact that I am a HUGE car enthusiast, and me trying to find accessible APIs online that would allow me to access information about cars.

Searching online gave me no results for open source or public APIs offering vehicle data, as most of them had been commercialized and only available B2B. I found a need, and decided to develop this web application.

I spent a few days aggregating various data and storing it into CSV files that I eventually pushed to a database. Following this, I built out a Python based web server to make the data accessible via a web interface and an API.

This project will always be public and open source, and I will continue to add to it. If anyone is interested in contributing, the source code is published via GitHub and I am happy to get anyone up to speed!
