API Basics

In addition to offering a vehicle report web service, this project exposes a public API endpoint that you can use to get data on any vehicle(s) for whatever project youâ€™re working on. Implementing the API is as simple as making an HTTP request to:

https://vehicleinfo.org/api/

and specifying the resource(s) you would like!

GET /year

A client can make a GET request for a specific year within range, and the server will return a JSON response of every vehicle for that year, including its individual attributes. Here is an example request:

https://vehicleinfo.org/api/2002

GET /year/make/

Next, a client can make a GET request for a specific year and make, and the server will return JSON response of a list of all vehicles produced by a manufacturer in that year. Here is another example request:

https://vehicleinfo.org/api/2002/acura

GET /year/make/model
