Welcome to the Vehicle Information Project

The Vehicle Information Project aims to make vehicle information public and accessible for anyone who needs it

Available Data

9300+ Vehicles

The project has aggregated over 9300 vehicles that were produced in the United States, ranging from cars, trucks, vans, and more.

60+ Makes

The dataset includes over 60 unique vehicle manufacturers from all over the world, such as Honda, BMW, and Chevrolet.

29 Model Years

Currently support vehicles produced from 1992 until 2020, with a growing range as more vehicles are added to the database

5+ Attributes

For each vehicle, there are a minimum of 5 attributes available, this includes but is not limited to available trims, MPG figures, engine size, and more.

Highlights

Vehicle Report Interface

This tool offers an online vehicle report interface where anyone can enter a vehicle's year, make, and model to access a comprehensive report. This report aggregates data from our in-house API, as well as data from the National Highway Traffic Safety Administration (NHTSA) about recalls and consumer complaints.

Open Source API

This tool also exposes an open source and public API endpoint that gives access to the data through an HTTP request to this web server. The API returns JSON formatted responses to requests for specific years, makes, or models of vehicles.

Mission

The purpose of this tool is to be a public resource for anyone interested in accessing a specific vehicle's information. The ongoing mission is to continually write robust scripts in order to add support for more vehicles and gather more data about the exisiting vehicles in the database.

This project is intended to be open source and public, so any and all contributions are always welcome!
