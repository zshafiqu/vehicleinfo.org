Welcome!

Welcome to the Vehicle Information project! This web application is a tool that can be used to access various details about vehicles. At the moment, the tool supports all vehicles (cars, trucks, vans, etc.) that were produced in the United States from 1992 until now. The dataset was built in-house, and includes over 9,300 vehicles with over 60 makes, and each of these 9,300 vehicles has at least five attributes available, images included.

Project Purpose

The purpose of this tool is to be a public resource for anyone interested in accessing a specific vehicleâ€™s information. Information can be accessed through the Vehicle Report interface on this website, or through an API call to the web server here.

Accessing the information through the web interface also compiles information about recalls and consumer complaints from NHTSA web resources and generates a comprehensive report.

Future

For each vehicle available, the tool has aggregated attributes such as body styles, images for each body style, available trims, and other details such as engine size, MPG, and more.

The ongoing goal for this project is to continually add support for more vehicles, and to write robust scripts gather more information for the existing vehicles in the database.

This project is intended to always be open source and public, so any contributions are always welcome!
